# 遊戲頁面應用

這是一個使用Node.js和Vite構建的遊戲頁面應用。

## 開發環境設置

### 安裝依賴

```bash
npm install
```

### 啟動開發服務器

```bash
npm run dev
```

開發服務器將啟動在 http://localhost:8080，並自動打開Games.html頁面。

## 生產環境部署

### 構建應用

```bash
npm run build
```

這將在`dist`目錄中生成靜態文件。

### 部署方法：使用Express服務器

1. 構建應用
2. 啟動Express服務器：

```bash
npm start
```
